# 🏠 Rent Prediction System (MLOps Project)

## 📌 Overview
This project predicts the rental price of residential properties based on features such as location, size, furnishing status, number of rooms, and amenities.

The goal is not just to build a machine learning model, but to design a system that can be extended into a full **MLOps pipeline** with automation, versioning, and deployment.

---

## 🎯 Objectives
- Build a regression model to predict house rent
- Implement a clean and reusable ML pipeline
- Prepare the project for MLOps integration (Databricks, MLflow, CI/CD)
- Enable real-time predictions using custom inputs

---

## 📊 Dataset
- **India House Rent Prediction Dataset (Kaggle)**
- Contains features like:
  - BHK (number of rooms)
  - Size (square footage)
  - Location (city, area type)
  - Furnishing status
  - Tenant preference
  - Number of bathrooms
